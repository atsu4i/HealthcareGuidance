# 🏥 保健指導シミュレーション - 使い方ガイド

このアプリケーションは、保健師の方が保健指導の練習をするためのシミュレーターです。

## 📋 必要な環境

### 初回のみ：Node.jsのインストール

アプリを起動する前に、**Node.js**をインストールする必要があります（初回のみ）。

1. 以下のサイトにアクセス：
   👉 https://nodejs.org/

2. **LTS版**（推奨版）をダウンロード
   - Windowsの場合：`.msi`ファイル
   - Macの場合：`.pkg`ファイル

3. ダウンロードしたファイルをダブルクリックしてインストール
   - 特別な設定は不要です。「次へ」を押していけばOKです

4. インストール完了後、パソコンを再起動

## 🚀 アプリの起動方法

### Windows の場合

1. `起動.bat` をダブルクリック
2. 黒い画面（コマンドプロンプト）が開きます
3. 初回起動時は、必要なファイルを自動でダウンロードします（数分かかります）
4. ブラウザが自動で開き、アプリが表示されます

### Mac の場合

1. `起動.command` をダブルクリック
2. 初回のみ、以下の確認が表示される場合があります：
   - 「開発元を確認できません」→「開く」をクリック
   - または、右クリック→「開く」→「開く」を選択
3. ターミナル（黒い画面）が開きます
4. 初回起動時は、必要なファイルを自動でダウンロードします（数分かかります）
5. ブラウザが自動で開き、アプリが表示されます

## 📱 使い方

### 1. Gemini APIキーの設定（初回のみ）

アプリを使用するには、Google Gemini APIキーが必要です。

1. 以下のサイトにアクセス：
   👉 https://aistudio.google.com/apikey

2. Googleアカウントでログイン

3. 「APIキーを作成」をクリック

4. 作成されたAPIキー（`AIza...`で始まる文字列）をコピー

5. アプリの画面で「設定」ボタンをクリック

6. 「Gemini APIキー」の欄に貼り付けて「保存」

### 2. シナリオの選択

1. ホーム画面で13種類のシナリオから選択できます：
   - 協力的・意欲的な対象者
   - 防衛的・否認傾向の対象者
   - 無関心・多忙な対象者
   - 知識はあるが実行しない対象者
   - など...

2. 「詳細確認」ボタンで対象者の詳細情報を確認できます

3. 「面談開始」ボタンで保健指導をスタート

### 3. 保健指導の練習

1. LINEのようなチャット形式で対話します

2. 対象者からの返答に対して、適切な指導を行います

3. 「面談終了」ボタンで終了できます

4. 「履歴」から過去の面談を確認できます

## 🛑 アプリの終了方法

### Windows の場合
- 黒い画面（コマンドプロンプト）を閉じる
- または、`Ctrl + C` キーを押す

### Mac の場合
- ターミナル（黒い画面）を閉じる
- または、`Command + Q` キーを押す

## ⚠️ トラブルシューティング

### ブラウザが自動で開かない場合
- 手動でブラウザを開き、以下のURLにアクセス：
  ```
  http://localhost:3000
  ```

### 「ポートが使用中」というエラーが出る場合
- すでにアプリが起動している可能性があります
- 一度すべてのブラウザとターミナル/コマンドプロンプトを閉じてから、再度起動してください

### Node.jsのインストールがうまくいかない場合
- パソコンを再起動してから、もう一度インストールを試してください
- それでもダメな場合は、古いバージョンのNode.jsがインストールされている可能性があります
- 「プログラムの追加と削除」（Windows）または「アプリケーション」（Mac）からNode.jsをアンインストールしてから、再度インストールしてください

### その他の問題
- パソコンを再起動してから、もう一度試してください
- それでも解決しない場合は、開発者に連絡してください

## 💡 Tips

- APIキーと設定は自動で保存されます（次回起動時も有効）
- 面談履歴も自動で保存されます
- 複数のシナリオを試して、様々な対象者への対応を練習できます
- 面談終了後に「フィードバック」機能で、指導内容の評価を受けることができます
